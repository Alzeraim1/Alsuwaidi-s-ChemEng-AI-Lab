Academic profile site for Ahmad Alsuwaidi — chemical reaction engineering,
CFD/DEM modeling, and machine learning for cleaner industrial processes.

## Files
- `index.html` — the website (open it to preview locally)
- `ahmad.jpg` — profile photo

## Publish for free (GitHub Pages)
1. Create a repo named `yourusername.github.io` (Public).
2. Upload `index.html` and `ahmad.jpg` together.
3. Settings → Pages → Branch: main → Save.
4. Live at https://yourusername.github.io within ~2 minutes.

## Or use Netlify Drop (no account needed)
Drag this whole folder onto https://app.netlify.com/drop — live instantly.
