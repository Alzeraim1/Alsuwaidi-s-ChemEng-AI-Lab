## Update on Netlify (drag-and-drop)
1. Go to app.netlify.com and open your site.
2. Open the Deploys tab.
3. Drag the WHOLE folder (all files above) onto the drop zone.
4. Live in a few seconds — same URL.

Tip: always drop the entire folder so the pages, photo, and Consultancy
link stay together.
